# pinout-cli 🚀

A blazing-fast, offline, terminal-based hardware component pinout lookup tool designed specifically for electronics and embedded systems engineering students. 

No more waiting for bloated 80-page manufacturer datasheets to download just to check which pin is Ground, VCC, or UART TX/RX while working under a lab microscope.

## Features
- **Zero Dependencies:** Built entirely with native Python 3 standard modules.
- **High-Contrast Terminal Art:** Clean ASCII-mapped chip footprints perfect for CLI environments.
- **Community Driven:** Easily customizable database—just add a dictionary key to include chips used in your local university labs.


## Installation
Clone this repository and run the automated installation script:

```bash
git clone [https://github.com/ceenorby/pinout-cli.git](https://github.com/ceenorby/pinout-cli.git)
cd pinout-cli
chmod +x setup.sh
./setup.sh
